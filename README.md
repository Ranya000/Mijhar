# مِجهر 🔍 — كاشف المخاطر للأفراد

منصّة تحلّل عقودك (**إيجار / تمويل / استثمار**) عبر **٦ وكلاء ذكاء اصطناعي**، وتكشف لك البنود الخطيرة والمخفية، تقارنها بالسوق، تحسب أثرها المالي، وتكتب لك رسالة اعتراض جاهزة — **قبل أن توقّع**.

> فريق: رانيا العتيبي (قائدة الفريق)، غدير، سمير، نورة العتيبي، رفعة البوعينين.

---

## الوكلاء الستة

| # | الوكيل | المهمة |
|---|--------|--------|
| 1 | **كاشف المخاطر** | يحدد كل بند خطير ويصنّفه (خطر/تنبيه/آمن) |
| 2 | **كاشف المخفي** | يكشف الثغرات والبنود الغامضة |
| 3 | **مقارنة السوق** | يقارن بنودك بالمعتاد في السوق السعودي |
| 4 | **النظرة المستقبلية** | يوضح ما سيحدث لك بعد التوقيع |
| 5 | **الأثر المالي** | يحلّل دخلك/ثروتك ويقرّر إن كان العقد يناسبك |
| 6 | **وكيل الاعتراض** | يكتب رسائل اعتراض قانونية جاهزة |

---

## هيكل المشروع

```
Mijhar/
├── backend/          خادم Node.js + Express + الوكلاء الستة
│   ├── src/
│   │   ├── agents/        الوكلاء (وكيل لكل ملف)
│   │   ├── data/          بيانات العقود النموذجية + تعريف الوكلاء
│   │   ├── lib/           محرّك الأثر المالي + عميل Claude + أدوات
│   │   ├── orchestrator.js   يجمع مخرجات الوكلاء في تقرير واحد
│   │   └── server.js         واجهة API
│   └── tests/         اختبارات Vitest
└── frontend/         واجهة React (Vite) موحّدة للعقود الثلاثة
    └── src/
        ├── App.jsx       التدفق: الرئيسية ← الرفع ← التحليل ← اللوحة
        ├── api.js        الاتصال بالباكند
        ├── theme.js      الألوان والثوابت والأنماط
        ├── ui.jsx        المكوّنات (أيقونات، بطاقات، مخططات)
        ├── panels.jsx    لوحات الوكلاء (مخاطر، مخفي، سوق، مستقبل، اعتراض)
        └── financial.jsx لوحة الأثر المالي + الحاسبة التفاعلية
```

## التشغيل

<div dir="ltr">

```bash
# 1) الباكند
cd backend
npm install
npm start          # يعمل على http://localhost:8787
npm test           # تشغيل الاختبارات (28 اختباراً)

# 2) الواجهة (في نافذة طرفية أخرى)
cd frontend
npm install
npm run dev        # يعمل على http://localhost:5173
```

</div>

ثم افتح `http://localhost:5173` في المتصفح. الواجهة تتصل بالباكند تلقائياً (proxy على `/api`).

### نقاط النهاية (API)

| الطريقة | المسار | الوظيفة |
|---------|--------|---------|
| `GET`  | `/api/health`  | فحص حالة الخادم |
| `GET`  | `/api/meta`    | الوكلاء + أنواع العقود + نصوص تجريبية |
| `POST` | `/api/analyze` | تحليل عقد: `{ contractType, text, financial? }` |

### الذكاء الاصطناعي (اختياري)

المشروع يعمل **بدون أي إعداد**: إن لم يوجد مفتاح API، يشتغل بـ**محرّك احتياطي** يعطي نتائج تحليل واقعية كاملة (مهم للعرض والتجربة).
لتفعيل التحليل الحقيقي بالذكاء الاصطناعي، انسخ `backend/.env.example` إلى `backend/.env` وضع مفتاح `ANTHROPIC_API_KEY`.

## التقنيات

- **الباكند:** Node.js، Express، بنية وكلاء متعددين (Multi-Agent).
- **الذكاء الاصطناعي:** Anthropic Claude عبر هندسة أوامر (Prompt Engineering).
- **الواجهة:** React (RTL) — قادمة.
- **الاختبارات:** Vitest.
