# عرض المحكمين — ملف واحد

`Majhar-Demo.html` هو **ملف عرض مستقل** لمشروع مجهر، جاهز لعرض المحكمين.

## كيف تفتحه
اضغط عليه مرتين (double-click) فيفتح بأي متصفح (Chrome / Safari / Edge). لا يحتاج
تنصيب ولا خادم ولا إنترنت — كل شيء بداخله.

## وش فيه
نفس الموقع الحقيقي بالضبط، لكن بتحليل نموذجي جاهز مدمج داخل الملف:
- الصفحة الرئيسية + تعريف الوكلاء الستة
- اختر نوع العقد (إيجار / تمويل / استثماري) واضغط «تجربة بعقد جاهز» ثم «حلّل العقد الآن»
- لوحة تحليل كاملة: كاشف المخاطر · كاشف المخفي · مقارنة السوق · النظرة المستقبلية ·
  الأثر المالي · وكيل الاعتراض

> الأرقام في هذا الملف نموذجية للعرض فقط. التحليل الحقيقي بالذكاء الاصطناعي يشتغل عند
> تشغيل الموقع كاملاً (الباكند + الواجهة) كما في README الرئيسي.

## كيف نبني هذا الملف من جديد
```bash
cd frontend
npm install
npm run build:demo          # ينتج frontend/dist-demo/index.html
```
ثم انسخه إلى `demo/Majhar-Demo.html`.
